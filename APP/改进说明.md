## 技术特性

### 支持的模型

1. **标准MLP** (Multi-Layer Perceptron)
   - 多层感知器神经网络
   - 支持自定义隐藏层结构
   - Adam优化器，早期停止

2. **Residual-MLP** (Residual Multi-Layer Perceptron)
   - 改进的MLP架构
   - 训练残差而非绝对值
   - 更好的温度变化捕捉

3. **Ridge回归**
   - 岭回归模型
   - L2正则化
   - 适合线性关系

4. **RandomForest**
   - 随机森林集成模型
   - 100棵决策树
   - 并行训练，特征重要性

### 特征工程

使用9个特征进行预测：

1. **电压** (voltage_v): 电池电压
2. **电流** (current_a): 充放电电流
3. **时间** (time_s): 时间序列
4. **环境温度** (ambient_temperature_c): 环境温度
5. **循环编号** (cycle_index): 充放电循环
6. **工况类型** (operation_type): 充电/放电标识
7-9. **滞后温度** (temperature_lag_1/2/3): 时间序列特征

### 性能指标

- **RMSE**: 均方根误差，衡量预测精度
- **MAE**: 平均绝对误差，衡量误差幅度
- **R²**: 决定系数，衡量模型解释能力
- **训练时间**: 模型训练耗时

## 安装和使用

### 快速开始

```bash
# 1. 安装依赖
pip install -r requirements.txt

# 2. 运行测试
python test_modules.py

# 3. 启动应用
python main.py
```

### 使用流程

1. **加载数据** → 选择.mat文件目录
2. **选择模型** → 选择适合的ML模型
3. **调整参数** → 设置超参数（可选）
4. **训练模型** → 开始训练并查看进度
5. **查看结果** → 分析性能指标和图表
6. **保存模型** → 保存训练好的模型
7. **交互预测** → 输入参数获取预测

## 扩展性

### 添加新模型

1. 在 `model_trainer.py` 中添加训练方法
2. 在 `config.py` 的 `MODELS` 列表中注册
3. 在 `gui.py` 中添加对应的UI逻辑

### 添加新特征

1. 修改 `config.py` 的 `FEATURE_COLS`
2. 在 `data_loader.py` 中添加特征创建逻辑
3. 更新 `dialogs.py` 中的输入框

### 自定义界面

1. 修改 `config.py` 中的颜色和字体配置
2. 调整 `gui.py` 中的布局
3. 添加新的对话框组件

## 已知限制

1. **数据格式**: 仅支持MATLAB .mat格式
2. **内存限制**: 大数据集可能消耗较多内存
3. **训练时间**: 复杂模型可能需要较长时间
4. **GPU支持**: 当前版本不支持GPU加速
