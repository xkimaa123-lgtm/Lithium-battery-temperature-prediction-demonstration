# 更新日志

## v2.1 (2026-06-15)

### 新增功能

#### 数据加载改进
- ✓ 支持选择单个.mat文件
- ✓ 支持选择数据目录
- ✓ 保留默认目录加载功能
- ✓ 显示加载的文件数量信息

#### 用户界面改进
- ✓ "选择文件"按钮 - 加载单个.mat文件
- ✓ "选择目录"按钮 - 批量加载目录中的所有.mat文件
- ✓ 数据标签显示文件数量信息

### 改进详情

#### 数据加载模块 (data_loader.py)
- 新增 `load_single_file()` 方法：支持加载单个.mat文件
- 新增 `load_directory()` 方法：支持加载目录中的所有.mat文件
- 新增 `_post_process()` 方法：数据后处理逻辑复用
- 改进 `load_data()` 方法：自动判断路径类型（文件或目录）
- 新增 `loaded_files` 属性：记录已加载的文件列表

#### 主窗口GUI (gui.py)
- 新增 `load_single_file()` 方法：打开文件选择对话框
- 新增 `load_directory()` 方法：打开目录选择对话框
- 新增 `_load_data_from_path()` 方法：通用数据加载逻辑
- 改进数据操作区域布局：分为两行按钮
- 改进数据标签显示：显示加载的文件数量

### 使用方式

#### 加载单个文件
1. 点击"选择文件"按钮
2. 在文件选择对话框中选择.mat文件
3. 点击"打开"
4. 系统自动加载并显示结果

#### 加载数据目录
1. 点击"选择目录"按钮
2. 在文件夹选择对话框中选择数据目录
3. 系统自动加载目录中所有.mat文件
4. 合并数据并显示统计信息

#### 使用默认目录
- 如果配置了默认数据目录，可以直接使用默认加载功能
