# 人工神经网络基础及应用课程-锂电池温度预测演示系统

## 系统概述

这是一个功能完整的锂电池温度预测演示系统，基于机器学习方法预测电池在充放电过程中的温度变化。

## 主要功能

### 1. 多种机器学习模型
- **标准MLP**: 多层感知器神经网络
- **Residual-MLP**: 残差MLP，改进的预测方法
- **Ridge回归**: 岭回归模型
- **RandomForest**: 随机森林模型

### 2. 超参数调优
- 自定义隐藏层神经元数量
- 可配置最大迭代次数
- 网格搜索自动寻找最优参数

### 3. 时间序列交叉验证
- 保持时间顺序的交叉验证
- 5折验证确保模型稳定性
- 提供详细的性能指标统计

### 4. 特征重要性分析
- 分析各特征对预测的贡献
- 可视化特征重要性排序

### 5. 数据验证和统计
- 数据范围验证
- 异常值检测
- 数据分布可视化
- 详细的统计报告

### 6. 模型管理
- 模型保存和加载
- 支持多种模型格式

### 7. 交互式预测
- 手动输入参数进行预测
- 实时预测结果显示

## 系统要求

- Python 3.7+
- 依赖包见 `requirements.txt`

## 安装步骤

1. 安装Python依赖：
```bash
pip install -r requirements.txt
```

2. 确保数据目录存在：
   - 默认数据路径: `E:\数据\data`
   - 数据格式: `.mat` 文件

## 使用方法

### 启动应用
```bash
python main.py
```

### 基本操作流程

1. **加载数据**
   - 点击"加载数据"按钮
   - 选择数据目录或使用默认路径
   - 等待数据加载完成

2. **选择模型**
   - 在左侧面板选择模型类型
   - 调整超参数（可选）

3. **训练模型**
   - 点击"开始训练"
   - 查看训练进度
   - 完成后查看结果

4. **查看结果**
   - 真实值vs预测值散点图
   - 误差分布直方图
   - 性能指标（RMSE, MAE, R²）

5. **多模型对比**
   - 点击"多模型对比"
   - 自动训练4种模型
   - 查看性能对比

6. **高级功能**
   - 交叉验证：评估模型稳定性
   - 超参数搜索：自动优化参数
   - 特征重要性：分析特征贡献

7. **交互预测**
   - 点击"交互预测"
   - 输入参数值
   - 获取预测结果

8. **保存/加载模型**
   - 点击"保存模型"保存当前模型
   - 点击"加载模型"加载已保存的模型

## 数据格式说明

系统期望的数据格式为MATLAB `.mat` 文件，包含以下结构：

```
battery_name
├── cycle[0]
│   ├── type: 'charge' | 'discharge'
│   └── data
│       ├── Time
│       ├── Voltage_measured
│       ├── Current_measured
│       ├── Temperature_measured
│       └── Ambient_temperature
├── cycle[1]
│   └── ...
```

## 特征说明

系统使用9个特征进行预测：

1. **voltage_v**: 电压 (V)
2. **current_a**: 电流 (A)
3. **time_s**: 时间 (s)
4. **ambient_temperature_c**: 环境温度 (°C)
5. **cycle_index**: 循环编号
6. **operation_type**: 工况类型 (1=充电, 0=放电)
7. **temperature_lag_1**: T(t-1) 前一时刻温度
8. **temperature_lag_2**: T(t-2) 前两时刻温度
9. **temperature_lag_3**: T(t-3) 前三时刻温度

## 性能指标

- **RMSE** (均方根误差): 衡量预测误差的平均幅度
- **MAE** (平均绝对误差): 衡量预测误差的平均绝对值
- **R²** (决定系数): 衡量模型解释数据变异的能力，越接近1越好

## 故障排除

### 1. 字体显示问题
如果中文显示异常，确保系统安装了以下字体：
- Microsoft YaHei (微软雅黑)
- SimHei (黑体)

### 2. 数据加载失败
- 检查数据目录路径是否正确
- 确保.mat文件格式正确
- 检查文件权限

### 3. 模型训练失败
- 检查数据是否包含NaN值
- 尝试减少隐藏层神经元数量
- 增加最大迭代次数

### 4. 内存不足
- 减少数据点数量
- 使用更小的模型
- 关闭其他应用程序

## 开发说明

### 添加新模型
在 `model_trainer.py` 中添加新的模型训练方法，并在 `MODELS` 列表中注册。

### 自定义特征
修改 `config.py` 中的 `FEATURE_COLS` 和 `data_loader.py` 中的特征创建逻辑。

### 界面定制
修改 `config.py` 中的 `COLORS` 和 `FONTS` 配置。
