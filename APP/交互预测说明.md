# 交互预测功能使用说明

## 功能概述

交互预测功能允许用户输入电池运行参数，实时预测下一时刻的电池温度。这是系统的核心功能之一，可用于：
- 电池温度监控
- 过热预警
- 充电策略优化
- 电池管理系统

## 前提条件

在使用交互预测功能之前，必须：
1. ✅ 加载数据（点击"选择文件"或"选择目录"）
2. ✅ 训练模型（点击"开始训练"）
3. ✅ 等待训练完成

## 使用步骤

### 步骤1：启动预测功能
- 点击主界面的"交互预测"按钮
- 系统会显示当前模型信息
- 自动打开预测窗口

### 步骤2：输入参数
在预测窗口中，需要输入以下9个参数：

| 参数 | 说明 | 合理范围 | 默认值 | 单位 |
|------|------|---------|--------|------|
| 电压 | 电池电压 | 2.0-5.0 | 3.7 | V |
| 电流 | 充放电电流 | -10.0-10.0 | 1.5 | A |
| 时间 | 时间点 | 0-10000 | 500 | s |
| 环境温度 | 环境温度 | -20.0-60.0 | 24 | °C |
| 循环编号 | 充放电循环编号 | 0-1000 | 50 | - |
| 工况类型 | 充电=1，放电=0 | 0或1 | 1 | - |
| T(t-1) | 前一时刻温度 | -20.0-80.0 | 30 | °C |
| T(t-2) | 前两时刻温度 | -20.0-80.0 | 29 | °C |
| T(t-3) | 前三时刻温度 | -20.0-80.0 | 28 | °C |

### 步骤3：执行预测
- 点击"预测温度"按钮
- 系统显示预测结果
- 结果自动保存到预测历史

### 步骤4：查看历史
- 预测历史显示在窗口底部
- 每次预测都会记录输入参数和预测结果
- 可以滚动查看更多历史记录

## 输入验证

系统会自动验证所有输入：

### 1. 非空验证
- 所有字段都必须填写
- 空值会提示警告

### 2. 数字验证
- 必须输入有效数字
- 非数字输入会提示错误

### 3. 范围验证
- 必须在合理范围内
- 超出范围会提示警告
- 范围显示在每个输入框旁边

### 错误提示示例
```
输入错误: 电压 必须在 2.0 到 5.0 之间
输入错误: 电流 必须是数字
警告: 请输入环境温度
```

```
- 循环编号: 50

- T(t-1): 40°C
```

```
结果：预测高温风险
```

## 技巧和最佳实践

### 1. 合理设置滞后温度
- T(t-1)、T(t-2)、T(t-3) 应该符合时间序列
- 通常 T(t-1) > T(t-2) > T(t-3)（充电升温）
- 或 T(t-1) < T(t-2) < T(t-3)（放电降温）

### 2. 注意电流符号
- 充电电流为正（如 +1.5A）
- 放电电流为负（如 -2.0A）
- 根据工况类型选择正确的符号

### 3. 循环编号的影响
- 新电池：循环编号小（如1-100）
- 老电池：循环编号大（如500+）
- 循环编号影响电池老化程度

### 4. 使用示例快速测试
- 首次使用点击"使用示例"
- 查看默认参数值
- 根据实际情况修改参数

## 常见问题解答

### Q1: 提示"请先训练模型"怎么办？
**原因**：还未训练模型就尝试预测

**解决方法**：
1. 返回主界面
2. 加载数据
3. 选择模型
4. 点击"开始训练"
5. 训练完成后再次尝试预测

### Q2: 预测结果不准确怎么办？
**原因**：输入参数不合理或模型未充分训练

**解决方法**：
1. 检查输入参数是否在合理范围内
2. 确保滞后温度符合时间序列规律
3. 尝试使用不同的模型类型
4. 使用超参数调优功能优化模型
5. 增加训练数据量

### Q3: 预测窗口卡住怎么办？
**原因**：计算量大或界面响应慢

**解决方法**：
1. 等待几秒钟
2. 如果仍然卡住，关闭窗口重新打开
3. 确保系统资源充足

### Q4: 如何保存预测结果？
**当前限制**：预测历史在窗口关闭后清空

**建议**：
1. 手动记录重要预测结果
2. 截图保存预测窗口
3. 使用打印功能导出历史

### Q5: 支持批量预测吗？
**当前版本**：不支持批量预测

**替代方案**：
1. 逐个输入参数进行预测
2. 使用"使用示例"快速填充
3. 重复使用相同的基础参数

## 高级功能

### 1. 参数敏感性分析
通过改变单个参数，观察预测结果变化：
```
基准：电压=3.7V → 预测温度=32°C
测试：电压=4.0V → 预测温度=33°C
结论：电压升高0.3V，温度升高1°C
```

### 2. 工况对比
对比充电和放电的温度差异：
```
充电（工况=1）：预测温度=35°C
放电（工况=0）：预测温度=30°C
差异：充电比放电高5°C
```

### 3. 老化影响分析
对比新旧电池的温度预测：
```
新电池（循环=50）：预测温度=32°C
旧电池（循环=500）：预测温度=34°C
差异：老化导致温度升高2°C
```
